# .learn — Guided Learning for FuzzySegment Pro

This folder contains short lessons and exercises so you can learn how Fuzzy C-Means and Granular Computing are applied.

Suggested order:

1. `01_intro.md` — conceptual overview and goals
2. `02_data_prep.md` — walkthrough of the data pipeline
3. `03_fuzzy_cmeans.md` — how fuzzy c-means works, hands-on
4. `04_streamlit.md` — how to extend the demo

Open the notebooks/ and follow step-by-step. As you complete lessons, update this file with notes.
